# Phase 3: The Feedback Loop

Measure, learn, and decide: expand winners or return to Phase 1.

---

## Objective

Use **creative scoring** to understand which specific elements drive performance — not just which ad won — and make a structured decision about what to do next.

---

## Step 1 — Creative Scoring

Implement a scoring system to isolate the variable driving performance. Don't just ask "which ad won?" Ask **"what element in the winning ad caused it to win?"**

### Core Metrics

| Metric | What It Tells You | Target Benchmark |
|--------|-------------------|------------------|
| Hook Rate (3-sec views / Impressions) | Whether the first 3 seconds stopped the scroll | > 25% |
| Hold Rate (25% views / 3-sec views) | Whether the middle held attention after the hook | > 40% |
| CTR (Link Click / Impressions) | Whether the offer + CTA converted interest to action | > 1.5% (cold), > 2.5% (warm) |
| ROAS | Efficiency of the full creative + offer combination | Define per client (typically 2-4x) |

Use the [Creative Scoring Template](../templates/creative-scoring-template.md) to log results.

---

## Step 2 — Scoring Visual Elements

When a winner emerges, go deeper. Use these questions to isolate the creative variable responsible:

### Element Isolation Questions

- [ ] Did ads with a **human face** in the first 2 seconds outperform those without?
- [ ] Did ads with a **problem-first hook** outperform benefit-first hooks?
- [ ] Did the **witty** copy register outperform the empathetic register?
- [ ] Did **static** outperform video — or vice versa — at this stage of the funnel?
- [ ] Did one **environment** (bathroom, kitchen, outdoor) generate more engagement?
- [ ] Did **short-form** (6s) outperform **mid-form** (15-30s)?
- [ ] Did any **CTA phrasing** consistently drive higher click-through?

### How to Isolate Variables

1. Compare ads that differ by **only one element** (same hook, different visual)
2. Use Meta's **A/B Test** tool for statistically valid comparisons
3. Use **Audience Segments** breakdown to see if different segments respond to different elements
4. Log findings in the Creative Learnings Log — this becomes your competitive advantage over time

---

## Step 3 — The Expand vs. Explore Decision

After scoring, make a structured decision:

### EXPAND — Scale the Winner

**When:** Winning ad has Hook Rate > 25% AND positive ROAS

| Action | Detail |
|--------|--------|
| Produce 50 more variants | New copy, new visuals, same hook — keep the winning element |
| Increase budget incrementally | No more than 20% every 48 hours |
| Add audience layers | Only after exhausting creative variation at current scale |
| Monitor for fatigue | Watch Hook Rate weekly — drop signals creative fatigue |

### EXPLORE — Return to Phase 1

**When:** Performance dips below threshold despite scaling attempts

| Signal | Action |
|--------|--------|
| Hook Rate drops below 25% | Creative fatigue — the audience has seen this angle too many times |
| ROAS declining despite budget stability | The hook is exhausted, not the audience |
| No clear winner after 7+ days | The current hooks aren't resonating — try new angles |

**Explore means:**
- Return to [Phase 1](phase-1-research-magic-brief.md) with **new angles** — not more of the same hook
- Use Creative Scoring data to inform which new direction to try
- Mine new customer language, find new competitor gaps
- Build a fresh Master Brief

---

## Creative Learnings Log

Maintain a running log of learnings across campaigns. Each entry should contain:

```markdown
## [Date] — [Campaign/Product] — [Finding]

**Winning Element:** [e.g., Problem-first hook with human face]
**Evidence:** Hook Rate 32% vs. 18% for benefit-first without face
**Audience Segment:** Performed best with New Audience
**Decision:** EXPAND — producing 20 more variants with this combination
**Next Review:** [Date + 7 days]
```

This log is your growing competitive advantage. Over time, it reveals patterns that no amount of best-practice reading can replicate.

---

## Deliverable

- Updated **Creative Scoring Template** with performance data
- **Creative Learnings Log** entry with isolated variable findings
- **Expand or Explore decision** with rationale documented
- If Explore: new Phase 1 brief direction identified

---

## Cadence

Run this review **weekly** for active campaigns. The feedback loop never stops — it's the engine that compounds creative performance over time.
