# Meta Ads Glossary

> **Source:** Jon Loomer | **Last Updated:** 2026-04-28 | **Status:** Complete

---

An A-to-Z reference of Meta advertising terms, each explained through practical usage context with links to the relevant SOP.

---

## A

**Advantage+ Audience** — Meta's algorithmic targeting system that uses your inputs as "suggestions" rather than hard boundaries. Automatically expands beyond any targeting you set when optimizing for conversions. See [General Targeting Approach](../02-targeting-audiences/general-targeting-approach.md).

**Advantage+ Creative** — A suite of enhancements Meta can apply to your ads (overlays, text improvements, visual touch-ups, animation, etc.) that only activate when they're predicted to improve performance. See [Ad Creation](../03-creative-ads/ad-creation.md).

**Advantage+ Placements** — Automatic placement selection that distributes ads across all eligible placements based on performance. The recommended default. See [Placements](../03-creative-ads/placements.md).

**Advantage+ Sales Campaign (ASC)** — Meta's simplified campaign type for purchase optimization. Uses algorithmic targeting, automatic placements, and CBO by default. See [Simplified Campaign Construction](../01-campaign-strategy-structure/simplified-campaign-construction.md).

**Andromeda** — Meta's ad retrieval engine. The first step in ad delivery that narrows millions of ads to a few thousand candidates for a given person. Not an algorithm. See [Meta Andromeda](../03-creative-ads/meta-andromeda.md).

**Attribution** — The rules for how conversions are credited to ads. Default is 7-day click, 1-day view. See [Attribution Optimization](../04-measurement-reporting/attribution-optimization.md).

**Audience Segments** — A reporting feature (Sales objective only) that breaks down results into Existing Customers, Engaged Audience, and New Audience. Defined in Advertising Settings. See [Audience Segments](../02-targeting-audiences/audience-segments.md).

## B

**Breakdown Effect** — A known reporting behavior where broken-down results don't sum to the campaign total. Expected, not a bug. See [Breakdown Effect](../04-measurement-reporting/breakdown-effect.md).

**Breakdowns** — Reporting dimensions (demographics, delivery, action, time) that let you slice results without changing campaign structure. See [Breakdowns](../04-measurement-reporting/breakdowns.md).

**Budget Cap (Existing Customer)** — A now-removed feature that let you cap budget spent on existing customers in ASC. No longer available. See [Budget Management](../01-campaign-strategy-structure/budget-management.md).

## C

**Campaign Budget Optimization (CBO)** — Budget set at the campaign level and distributed across ad sets automatically. Default for Advantage+ campaigns. See [Budget Management](../01-campaign-strategy-structure/budget-management.md).

**Conversions API (CAPI)** — Server-side tracking that sends events directly to Meta, bypassing browser limitations. Complements the Meta Pixel. See [Conversion Tracking](../04-measurement-reporting/conversion-tracking.md).

**Creative Diversification** — Providing Meta with meaningfully different creative assets (formats, visuals, text, angles) so the delivery system can match the right version to the right person. The primary lever for leveraging Andromeda. See [Meta Andromeda](../03-creative-ads/meta-andromeda.md).

**Creative Fatigue** — When an audience has seen the same creative too many times, leading to declining engagement. Solved by adding diverse new creative. See [Creative Testing](../03-creative-ads/creative-testing.md).

**Creative Testing** — Meta's built-in tool for introducing new ads within an existing ad set with a dedicated budget portion. See [Creative Testing](../03-creative-ads/creative-testing.md).

## D

**Dynamic Creative** — A feature that lets you provide multiple images, videos, and text variations within a single ad. Meta assembles combinations dynamically. Being evolved into Flexible Format. See [Ad Creation](../03-creative-ads/ad-creation.md).

## E

**Engaged Audience** — An Audience Segment representing people who interacted with your business but haven't purchased. Defined using custom audiences (website visitors, email list). See [Audience Segments](../02-targeting-audiences/audience-segments.md).

**Existing Customers** — An Audience Segment representing people who have purchased from you. Defined using purchase event CAs and buyer email lists. See [Audience Segments](../02-targeting-audiences/audience-segments.md).

## F

**Flexible Format** — An ad format that lets you provide up to 10 images or videos within a single ad. Meta generates carousels and selects the best creative per person and placement. See [Ad Creation](../03-creative-ads/ad-creation.md).

## G

**GEM** — Meta's ad ranking system. While Andromeda handles retrieval (picking the shortlist), GEM ranks the final winning ads that get shown. See [Meta Andromeda](../03-creative-ads/meta-andromeda.md).

## L

**Learning Phase** — The initial period after launching or significantly editing a campaign where Meta's algorithm is gathering data. Typically needs about 50 conversions to exit. Avoid making changes during this phase. See [Budget Management](../01-campaign-strategy-structure/budget-management.md).

**Lookalike Audiences** — Audiences modeled after a source audience (e.g., purchasers). Now automatically expanded when using Advantage+ targeting. See [Detailed Targeting and Lookalike Audiences](../02-targeting-audiences/detailed-targeting-and-lookalike-audiences.md).

## M

**Meta Pixel** — JavaScript code installed on your website that tracks user actions (events) for conversion tracking and optimization. See [Conversion Tracking](../04-measurement-reporting/conversion-tracking.md).

## N

**New Audience** — An Audience Segment representing people who are not in your Existing Customers or Engaged Audience — effectively prospecting. See [Audience Segments](../02-targeting-audiences/audience-segments.md).

## R

**Remarketing** — Showing ads to people who already interacted with your business. Now happens naturally with algorithmic targeting — separate remarketing campaigns are no longer recommended. See [Remarketing](../02-targeting-audiences/remarketing.md).

**Retrieval** — The first step of ad delivery where the system narrows millions of ads to a shortlist. Andromeda is Meta's current retrieval engine. See [Meta Andromeda](../03-creative-ads/meta-andromeda.md).

## V

**Value Rules** — Bid multipliers that tell Meta which conversions are worth more, based on audience groups or demographics. Steers delivery without restricting audiences. See [Value Rules](../01-campaign-strategy-structure/value-rules.md).

**View-Through Conversion** — A conversion counted when someone SAW your ad (didn't click) and later converted within the attribution window. Can inflate reported results. See [Attribution Optimization](../04-measurement-reporting/attribution-optimization.md).
