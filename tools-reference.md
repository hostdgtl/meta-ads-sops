## Evaluation Criteria for New Tools

Before adding any tool to the approved list:

1. **Does it replace manual work?** — Must save measurable time in the 3-phase process
2. **Does it integrate with existing stack?** — Must work with Claude, Shopify, Meta Ads Manager
3. **Is the cost justified?** — Must show ROI within one campaign cycle
4. **Is there a free alternative?** — Check open-source or built-in Meta tools first
5. **Does it lock us in?** — Prefer tools with data export and API access
