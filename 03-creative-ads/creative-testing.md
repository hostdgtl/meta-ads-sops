# Creative Testing

> **Source:** Jon Loomer | **Last Updated:** 2026-04-28 | **Status:** Complete

---

## What It Is

A structured approach to testing new ad creative within existing campaigns. Meta's built-in Creative Testing tool lets you introduce new ads with a dedicated budget portion, then return to normal delivery after the test.

## Why It Matters

Adding new creative to a running campaign may not get fair distribution — Meta tends to favor proven performers. The Creative Testing tool solves this by guaranteeing budget for new creative, giving you reliable data on what works.

## How It Works

1. Within an existing ad set, launch new ads using the Creative Testing tool
2. Dedicate a set amount of budget to those test ads
3. Meta runs the test for a defined period with guaranteed delivery
4. When completed, the ads integrate into normal delivery
5. Winning creative gets full delivery; underperformers can be paused

## Common Mistakes

- Adding new ads to a running ad set and expecting equal distribution (Meta favors existing winners)
- Judging creative based on surface metrics only (clicks, impressions) instead of conversions
- Not using Audience Segments breakdown to see if different creatives perform differently across segments
- Testing too many variables at once — change one thing at a time
- Ending tests too early before enough data accumulates

## How to Apply It

1. Build your test creative following the Ad Creation SOP
2. Open your existing ad set in Ads Manager
3. Use the Creative Testing tool to add new ads with a dedicated budget
4. Let the test run for at least 3-7 days (longer for lower budgets)
5. Review results by conversion metrics, not vanity metrics
6. Break down by Audience Segments to understand which creative works for which group
7. Keep winners, pause losers, iterate

## When to Make Exceptions

- If you're launching a brand new campaign, you don't need the Creative Testing tool — just load diverse creative from the start
- Very low budgets may not generate enough data for conclusive tests
